# Симулятор пандуса — ДБН В.2.2-40:2018

Інтерактивний інструмент для перевірки геометрії пандуса на відповідність вимогам **ДБН В.2.2-40:2018 (Зміна №3)** — українського будівельного нормативу з інклюзивності будівель і споруд.

![Скріншот симулятора](https://raw.githubusercontent.com/wiki/placeholder.png)

## Можливості

- **Миттєвий вердикт** — зелений / бурштиновий / червоний індикатор відповідності нормі
- **11 перевірок** за нормою: ухил, висота маршу, ширина, поручні, бортики, покриття тощо
- **Два режими** — нове будівництво та реконструкція (пом'якшені вимоги)
- **Схеми у реальному масштабі** — профіль (вид збоку) та план (вид зверху)
- **Інтерактивні повзунки** з кольоровими зонами допустимості
- **Автоматична розбивка** на марші з проміжними площадками при H > 0,80 м
- **Навчальний тур** для першого ознайомлення

## Запуск

Просто відкрийте файл `pandus-simulator.html` у браузері.

> **Примітка:** потрібне інтернет-з'єднання для завантаження React, ReactDOM та Babel з CDN.

## Структура проекту

```
pandus-simulator.html   — головна HTML-сторінка зі стилями
norms.js                — нормативне ядро: геометрія та перевірки за ДБН
controls.jsx            — UI-компоненти: слайдери, перемикачі, вкладки
views.jsx               — SVG-візуалізації: профіль та план пандуса
app.jsx                 — головний React-компонент з логікою стану
```

## Технології

- **React 18** (з CDN)
- **Babel** (in-browser JSX трансформація)
- **Vanilla CSS** з кастомними CSS-змінними
- **SVG** для геометричних креслень
